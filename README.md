# TheCafeOrderingSystem

Initial upgraded Cafe Ordering System that provides an efficient, user-friendly product by giving the users a graphical user interface for managing menu items, placing orders, and generating bills.

Key features of the program:
     Good menu management, allwoing items to be added, removed, and marked as available or unavailable.
     Clear user interface allowing the users to navigate around efficiently
     Structured ordering
     Automatic subtotal, tax and total calculation for orders
     Real-time ordering updates
